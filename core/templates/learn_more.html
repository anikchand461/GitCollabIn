üîê Adding Your GitHub Token
‚ñ∂

For advanced features like viewing forks and opening issues, a personal access token is recommended.

How to Generate:

  1. Go to GitHub Tokens
  2. Click "Generate new token (classic)"
  3. Select scopes: repo, read:user, user:email

How to Add:

  • Go to your GitCollab Profile > Edit.
  • Paste the token in the token field.
  • It's securely masked and stored.
